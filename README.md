# React + Tailwind CSS+APIで天気アプリケーション作成

## プロダクトの紹介
- OpenWeatherAPIとTailwindCSSを使って、複数都市の天気を一覧表示するもの

## 工夫したところ
- 使ったことがなかったTailwindCSS+OpenWeatherAPIに挑戦してみたところ
- APIキーを.envに記載して、.gitignoreしたこと

## 苦戦したところ

- 原因不明のままなので、後でもう少し原因を探ってみようと思います
